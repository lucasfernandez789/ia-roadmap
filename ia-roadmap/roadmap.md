# Roadmap de Aprendizaje - IA

## Contexto y Objetivos

- Usuario con experiencia en frontend (React, Vite, Tailwind, Cypress, etc.).
- Matemática: buena base en discreta, estadística y matrices, aunque con dudas puntuales.
- Objetivo: aprender Python y herramientas de IA/ML desde lo básico, con enfoque práctico y control de versiones.
- Preferencia: ejercicios cortos por tema + mini-proyectos integradores, usando Git y Jupyter desde el inicio.

---

## Avances

### 1. Instalación y setup

- Python instalado correctamente.
- Jupyter Notebook instalado y funcionando (usando `python -m notebook` en PowerShell).
- Repo `ia-roadmap` creado y estructurado.
- `.gitignore` configurado para ignorar archivos temporales, entornos y datasets pesados.

### 2. Python Básico

- Repaso de variables, tipos de datos, operaciones básicas.
- Diferencia entre listas y diccionarios:
  - Lista: colección ordenada de elementos.
  - Diccionario: colección de pares clave:valor.
- Ejercicios realizados:
  - Definición y operaciones con variables numéricas.
  - Creación y recorrido de listas.
  - Creación, acceso y recorrido de diccionarios.
- Dudas resueltas:
  - Diferencia entre lista y diccionario.
  - Acceso correcto a valores en un diccionario dentro de un ciclo.
  - Uso de claves y valores en ciclos for.

### 3. Reflexiones y próximos pasos

- El usuario comprendió la sintaxis básica y la diferencia entre estructuras.
- Próximo objetivo: practicar más con listas y diccionarios, luego avanzar a Numpy y Pandas.
